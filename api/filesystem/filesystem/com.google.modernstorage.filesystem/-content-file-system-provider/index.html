A FileSystemProvider for content:// scheme Uris.

Because of the way ART works, it's not possible to get an instance of this class the way you would on a JVM (using FileSystems.getFileSystem). Instead you should explicitly create an instance of this class (i.e.: fileSystemProvider = ContentFileSystemProvider(context)).
